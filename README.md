# cli 3.0

```bash
## Clone project
git clone git@github.com:782042369/love-baby.git

## Install dependencies
npm install

## 建议不要用cnpm  安装有各种诡异的bug 可以通过如下操作解决npm速度慢的问题
npm install --registry=https://registry.npm.taobao.org

## serve with hot reload at localhost:9528
npm run dev

## build for production with minification
npm run build

## postcss-px-to-viewport 设置说明  https://segmentfault.com/a/1190000010947054
viewportWidth 375 设计稿的宽度
unitPrecision 4 px转成vw、vh后小数点保留的位数
minPixelValue 1 不转化为vw的最小px值
```
